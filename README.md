# Weather Dashboard MCP Assignment

This is a small weather dashboard project built for the MCP and Skills Across the Software Lifecycle assignment.

## Features
- Search for a city
- Display current weather
- Show temperature
- Show weather condition
- Show humidity
- Show wind speed

## Tech Stack
- React
- Vite
- JavaScript
- OpenWeather API

## Project Status
- Basic UI created
- Search flow connected
- Weather API integration added
- Final API testing pending key activation
